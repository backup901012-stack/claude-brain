### 密鑰存放位置（三層）

| 環境 | 存放位置 | 用途 |
|:---|:---|:---|
| GitHub Actions | Repo → Settings → Secrets | 自動化 workflow |
| Cloudflare Workers | Workers Secrets | D1、API 端點 |
| 本機開發 | `~/.env.shared` 或各專案 `.env`（不進 git） | 本機測試 |

### 密鑰輪換 SOP（每次申請或輪換密鑰時必須四步全做）

```
申請/輪換 API Key
    ├── 1. 更新本機 .env                ← 本機能跑
    ├── 2. 更新 GitHub Secrets          ← Actions 能跑（不能漏！）
    ├── 3. 更新 Workers Secrets         ← Cloudflare 能跑（如適用）
    └── 4. 記錄到密鑰總表               ← 追蹤管理
```

**四步都做完才算完成。漏任何一步會導致對應環境跑不動。**
