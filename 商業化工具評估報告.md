## 七、各業務線的工具對應

### 命理諮詢（單次收費）

```
客戶下單 → Stripe 收款
    → Notion 記錄客戶資料
    → 命理系統 session 生成報告（PDF+PPT）
    → 品控檢核
    → Gmail 發送報告 + Google Calendar 建行事曆
    → DocuSign 簽免責聲明（可選）
```

### 宏觀日報（訂閱制）

```
客戶訂閱 → Stripe Billing 自動扣款
    → Brevo 管理訂閱者名單
    → GitHub Actions 每天 05:00 收集數據
    → 07:30 自動發送報告給所有訂閱者
    → Google Analytics 追蹤開信率
```

### 股票量化（訂閱制 + API）

```
客戶訂閱 → Stripe Billing 自動扣款
    → Cloudflare Workers API 提供數據查詢
    → Dashboard（Cloudflare Pages）展示即時數據
    → Stripe 按 API 呼叫次數計費（進階方案）
```

### 賽馬分析（單場/訂閱制）

```
賽前分析 → GitHub Actions 自動執行
    → 報告生成（PDF）
    → Gmail/Brevo 發送給付費用戶
    → Stripe 收款
```
